# North-star stroke order (for authors)

Each lesson is deconstructed from a simple **silhouette** goal: big shapes first, details last. Step counts: **cat 7**, **car 6**, **house 6**.

## Cat (friendly sitting cat)

1. Light guide circle (head).
2. Final head circle.
3. Ears (curved, on top third of head).
4. Face: eyes, nose, smile.
5. Body oval (neck gap).
6. Four paws.
7. Tail + whiskers.

## Car (side view, cartoon)

1. Light guide block for body + cabin area.
2. Solid body + cabin shape.
3. Wheels (equal size, grounded).
4. Windows.
5. Ground / road line.
6. Lights + bumper.

## House (front view)

1. Light guide square (walls).
2. Neat wall square.
3. Roof triangle (same width as walls).
4. Door + knob.
5. Windows (+ cross panes).
6. Chimney + path.
