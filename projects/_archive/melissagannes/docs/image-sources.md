# Image and reference sources (StepDraw Jr)

Use this file to record **public domain**, **CC0**, or otherwise **cleared** material you used when planning proportions, poses, or exports from Figma / Illustrator.

Do **not** paste copyrighted “how to draw” tutorial text into the app. Use references only to inform **your own** vectors and copy.

## Policy

- Prefer **public domain** or **CC0** references (photos, old diagrams, museum open access).
- Read the **license on each file** before you rely on it for a shipped graphic.
- For anything you **export from Figma or Illustrator**, you own the vector work as long as you did not trace someone else’s copyrighted illustration without permission.

## Where shipped art lives

- Optional per-step SVGs: [`base_mvp/public/steps/`](../base_mvp/public/steps/README.md) — drop Figma/Illustrator exports as `/<lessonId>/<stepIndex>.svg` (e.g. `cat/0.svg` …). If a file is missing, the app falls back to inline diagrams in code.
